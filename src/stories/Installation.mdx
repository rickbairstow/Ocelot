# Installation

## Prerequisites

- Vue 3.5+
- Tailwind CSS v4
- Node.js 20+

## Install

```bash
npm install ocelot-ui
```

Optional feature dependencies:

```bash
# Table
npm install @tanstack/vue-table

# LightboxImage
npm install photoswipe

# Video
npm install plyr

# QrCode
npm install qrcode
```

Install only the packages for the components you actually use. OUI warns in development if `Table`, `LightboxImage`, `Video`, or `QrCode` is mounted without its matching dependency.

## Setup

### 1. Configure Tailwind

Import the OUI Tailwind plugin in your main CSS file. It adds the `@source` scan for OUI classes, design tokens, and the dark mode variant bridge:

```css
/* your main CSS file */
@import "tailwindcss";
@import "ocelot-ui/tailwind.css";
@custom-variant dark (&:where(.dark, .dark *));
```

### 2. Import Vue transition styles

```js
// main.js / main.ts
import 'ocelot-ui/style.css'
```

### 3. Use components

Import components directly — fully tree-shakeable:

```js
import { Button, Dialog, Toast } from 'ocelot-ui'
```

Or register globally:

```js
// main.js / main.ts
import { createApp } from 'vue'
import { Button, Input, Dialog } from 'ocelot-ui'
import 'ocelot-ui/style.css'
import App from './App.vue'

const app = createApp(App)
app.component('Button', Button)
app.component('Input', Input)
app.component('Dialog', Dialog)
app.mount('#app')
```

### Table notes

`Table` depends on `@tanstack/vue-table` and includes a built-in settings menu for density, column visibility, and pinning. On smaller screens, you can keep the table scrollable or switch to stacked cards with `mobileLayout="card"`.

## Icons

OUI uses [Tabler Icons](https://tabler.io/icons). A built-in set of common icons is bundled. To use additional icons, register them once at app startup:

```js
import { registerIcons } from 'ocelot-ui'
import { IconBrandGithub, IconStarFilled } from '@tabler/icons-vue'

registerIcons({ IconBrandGithub, IconStarFilled })
```

Then reference them by name (without the `Icon` prefix) or pass the component directly:

```vue
<Icon icon="BrandGithub" />
<Button :icon="IconBrandGithub">GitHub</Button>
```

---

## Dark mode

Dark mode is toggled via the `.dark` class on a parent element (typically `<html>`):

```html
<html class="dark">
  ...
</html>
```

---

## Dialog portal target

The `Dialog` component uses Vue's `<Teleport>` to render outside the component tree.
Add a `#portal-target` element to your `index.html`:

```html
<div id="app"></div>
<div id="portal-target"></div>
```

---

## Design tokens

Override OUI's CSS custom properties to customise appearance without a Tailwind rebuild:

```css
:root {
    --oui-radius-lg: 0.75rem;
}
```

See `ocelot-ui/tailwind.css` for the full list of available tokens.
