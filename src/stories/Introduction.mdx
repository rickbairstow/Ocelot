# Ocelot UI

Ocelot UI (OUI) is a Vue 3 component library built with Tailwind CSS v4. It provides accessible, themeable components optimised for modern web applications.

## Components

| Component | Description |
|-----------|-------------|
| Accordion / AccordionGroup | Collapsible content panels with exclusive-open group mode |
| Avatar / AvatarGroup | User avatars with image, initials, and icon fallbacks |
| Badge | Inline status labels with colour and size variants |
| Banner | Inline feedback messages (info, success, warning, error) |
| Breadcrumb | Navigation breadcrumb trail |
| Button | Action buttons with variants, icons, loading state |
| ButtonGroup | Grouped set of related buttons |
| Card | Content containers with optional image, actions, badges |
| CodeBlock | Syntax-highlighted code display |
| Dialog | Accessible modal with focus trapping and portal rendering |
| Divider | Horizontal/vertical separators with optional label |
| DropdownMenu / DropdownMenuTrigger / DropdownMenuContent / DropdownMenuItem / DropdownMenuCheckboxItem / DropdownMenuLabel / DropdownMenuSeparator | Composable dropdown menu with keyboard navigation |
| EmptyState | Placeholder UI for empty data states |
| FloatingPanel | Positioned overlay panel |
| Heading | Semantic heading with size control |
| Icon | Tabler icon wrapper with registry and tree-shaking |
| Image / LightboxImage | Responsive image with optional lightbox |
| Input | Text field with label, validation, and icon support |
| Label | Form label |
| Loader | Animated loading indicator |
| Pagination | Page navigation controls |
| Placeholder | Skeleton placeholder for loading states |
| Popover | Floating content panel anchored to a trigger |
| Progress | Progress bar with value and label |
| QrCode | QR code generator |
| Quote | Styled blockquote |
| ReadMore | Truncated text with expand toggle |
| Scrim | Overlay backdrop |
| Sidebar | Collapsible navigation sidebar |
| StatCard | Key metric display card |
| Stepper | Multi-step process indicator |
| Tabs / Tab / TabList / TabPanel | Accessible tabbed interface |
| Timeline | Vertical chronological event list |
| Toast | Notification toasts via `useToast` composable |
| Tooltip | Contextual hover tooltip |
| Video | Embedded video component |

See the **Installation** page for setup instructions.
