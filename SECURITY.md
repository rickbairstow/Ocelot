# Security Policy

## Supported Versions

We currently provide security updates for the latest version of this project.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by contacting the project maintainer directly.

Please include:
- A description of the issue
- Steps to reproduce it
- Any potential impact
- Suggested remediation, if known

Please do **not** open a public GitHub issue for security vulnerabilities.

We will aim to acknowledge reports within a reasonable timeframe and address valid issues as appropriate.
